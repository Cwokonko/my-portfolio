Projects

As an ECE major I enjoy navigating the boundary between the hardware and software realm. I'm really passionate about physical computing as it enables me to blur the lines between the digital and physical world. Below are a few projects that I've worked on that allowed to put my passion to practice.

    Machine Learning Handwriting Recognition Systems:

  • Using MNIST digit data set, implemented five distinct handwriting recognition algorithms that leveraged various data structures, algorithms, and programming paradigms such as OOP Concurrent Programming, Generic Programming, Multithreading, Linear and Binary Search, and Graph and Table data structures.
  • Profiled memory usage & execution time using flamegraphs to analyze and optimize the training and recognition of a 60K dataset and 10K testing dataset to execute within a second with 91% accuracy.

    • Human-Machine Interface:

  • Designed and Programmed an embedded system capable of conducting fatigue analysis testing and remotely controlling electronics via a muscle sensor using an Arduino, Raspberry Pi, and Analog sensors. Executed communication protocols using Radio Frequency Transceivers

    • Smart Home Systems:

  • Prototyped and coded a temperature/humidity system to detect when an environment exceeded a threshold temperature or humidity value. Helped reduce frequency of common cold symptoms for users.
  • Designed and prototyped an Automated night light using RGB LEDs, Photo resistors, and Real Time Clock. Implemented a Set data structure used to hold the color values for LEDs and Perform data analysis on light usage within a room. Ultimately led to improved energy efficiency, better light usage, and a decrease in household electric bill.
  • Implemented a portable and fast queuing system for an LED Light Display that uses a keypad matrix to queue up a set of RGB values to output to the LEDs. Implemented a Queue data structure from scratch.

    • Ultrasonic Robot Car:

  • Built an avoidance robot car and programmed digital logic controls to interact with circuits and sensors such as H-bridges, potentiometers ultrasonic sensors, DC & servo motors.
